# smiley-wip
my video game. I'm essentially making a game engine entirely from scratch in C++

this game features custom-made UI interaction with custom windows and mouse position in a dynamic vector-coordinate plane, json save file parsing, cpp library inclusions and additions, custom art and music (not yet), and other intricacies not commonly seen in cpp; all things except the libraries have been created by me alone so far.

Please (git)IGNORE the mess right now. this is the very first game i've ever made, and i am realizing better architectures and developing new ideas to implement as i am coding it, so it is not perfect. i will be converting main.cpp to refer to all pointer objects as smart pointers to significantly reduce the memory on the stack during execution. i have yet to figure out how i can turn this around and sell it to others and have it be playable on PC: i thought about including a package that essentially runs in the user's shell, checking to see if the user has any cpp compiler installed, if not then installing one, then launching the game's code files to run in said compiler; but that may be a very taxing approach.
