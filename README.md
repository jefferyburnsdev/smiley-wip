# smiley-wip
my video game. I'm essentially making a game engine entirely from scratch in C++

this game features custom-made UI interaction with custom windows and mouse position in a dynamic vector-coordinate plane, json save file parsing, cpp library inclusions and additions, custom art and music (not yet), and other intricacies not commonly seen in cpp; all things except the libraries have been created by me alone so far.
